## Tracking spreadsheet shape

After sending each email, log in a simple sheet (Notion table works too):

| Hotel | Contact name | Email | Date sent | Response | Status | Notes |
|-------|-------------|-------|-----------|----------|--------|-------|
| | | | | | sent / pending / declined / negotiating / locked | |

Statuses:
- **sent** - first email out, no reply yet
- **pending** - they acknowledged, working on internal answer
- **negotiating** - terms on the table
- **declined** - they passed, log reason
- **locked** - room block confirmed, add to Partners list on zaostock.com

4. Loop in Iman / Zaal-shared inbox for booking coordination
